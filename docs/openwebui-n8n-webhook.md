# Openwebui - n8n - webhook

* First in the N8N flow, place a webhook trigger configured with the post method
* Open Open WebUI in your browser.
* Go to ⚙️ Admin Settings → Functions → click on **New Function**
* Delete the example code
* Set the name and description of the function
* Copy and paste the contents of the [file](https://raw.githubusercontent.com/tecno-consultores/llm-lab/refs/heads/main/docs/n8n%20Pipe%20Function.txt)
* Click **Save** and when prompted, click on **confirm**.
* Now click on the gear icon to set the function parameters.
* Set **N8N Url** and **N8N Bearer Token**
* Click **Save** and enable the function
