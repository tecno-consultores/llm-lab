# Openwebui - ollama

To use OpenWebUI as UI for ollama:

* Open Open WebUI in your browser.
* Go to ⚙️ Admin Settings → Connections
* Enable **Ollama API*
* Now click on the gear icon to set the api parameters:
  - URL: http://ollama:11434
  - Click the refresh icon next to the URL field to verify the connection.
* Click **Save**
* Enable the connection to Ollama
